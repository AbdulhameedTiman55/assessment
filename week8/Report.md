# Week 8 – Security Testing

## Overview
This week focused on identifying and mitigating security vulnerabilities using manual and automated techniques.

## Lab 8A – Manual Security Code Review
A SQL Injection vulnerability was identified where user input was directly embedded into a SQL query.
The issue was mitigated using parameterised ORM queries.

## Lab 8B – Static Analysis Security Testing
Snyk was used to perform static code analysis.
Identified vulnerabilities were reviewed, prioritised, and documented.

## Conclusion
Combining manual and automated security testing provides improved coverage and early risk detection.
